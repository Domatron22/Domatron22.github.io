Minecraft Server

August 20, 2022 by Domatron

We have a minecraft server with multiple instance of minecraft running. Here is a description of the servers that we have operational (All servers are Java Edition)

  • 1.19 Vanilla
  • 1.19 Vanilla with QoL Datapacks
  • 1.19 Vanilla with a Space Exploration Datapack
  • 1.18.1 Modded Server (Special Occasions)
1.19 Vanilla
This instance is well, vanilla. This server is attempting to be a longstanding map. We hope to update it when new version come out, and have people come on whenever.
1.19 Vanilla with Quality of Life Datapacks
This instance will require you to download some client side resource packs to get the full experience. This instance uses the following datapacks
  • Incendium
  • Nullscape
  • Terralith
  • Swords and Strongholds
  • Dungeon Now Loading 17%
The only packs that you will need to download a resource pack for are Swords and Strongholds, and Dungeons Now Loading.
1.19 Vanilla with a Space Exploration Datapack
Space, the final frontier... This instance uses a data pack called The Expansion
This pack requires a little bit of reading before you are able to just launch into space. Take a look at the site linked above to get started and grab the resource pack. Before you know it, you'll be crusing in the cosmos
1.18.1 Modded Server (Special Occasions)
We have a custom mod pack thjat we like to use for those days where regular minecraft just isnt cutting it anymore, and we want some harder stuff to deal with.

Joining the GAANG

Before continuing, I must say, that there is no cost whatsoever to join the Server. As long as we have the room you are more than welcome to join. At this point in time, all instances except for the modded server are running. Each has a player cap of 20, depending on how the server can handle this, we might be able to up the count, or minimize others based off of participation

If you would like to join, click on the discord link (The discord icon to the right) and that will direct you to the new discord server. After proving that you are not a bot, we will allow you access to the server if there is room

Some general rules

  • No griefing please (If you want an anarchy server, we can do this, just not on the ones mentioned above)
  • Profanity IS allowed
  • If you decide to invite friends, please invite them to the discord server, just so we can keep track of players, plus we will need to whitelist you

Finally, Thank you to the devs of any of these datapacks, your work is much appreciated, and the community is very happy with your contributions.
